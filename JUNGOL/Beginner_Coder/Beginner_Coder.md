  - 약수구하기

    - N = A * B이므로 약수는 짝을 이루어 나온다. 따라서 아래와 같이 시간복잡도를 줄일 수 있다.

      ```python
      n = int(input())
      
      divisorsList = []
      
      for i in range(1, int(n**(1/2)) + 1):
          if (n % i == 0):
              divisorsList.append(i) 
              if ( (i**2) != n) : 
                  divisorsList.append(n // i)
      
      divisorsList.sort()
      
      for r in divisorsList:
          print(r, end = " ")
      ```
